## 1. 系统概述

本系统用于招聘团队日常管理候选人简历、追踪面试流程、填写日报周报、查看统计数据。

**核心流程**：上传简历 → 识别信息 → 管道阶段推进 → 日报/周报自动汇总

**两种角色**：
- **管理员**：可查看所有实习生的数据，管理用户账号
- **实习生**：只能查看和操作自己的数据

**访问地址**：`http://8.149.143.126`

## 3. 首页（工作台）

### 3.1 上传简历
首页中央是 PDF 拖拽区，支持：
- **拖拽** PDF 文件到虚线框中
- 或**点击**虚线框选择文件
- 支持**批量上传**，多个文件依次解析

解析完成后出现**审核面板**，显示 AI 提取的字段（姓名、手机、学历、应届生等）。确认无误后选择「推荐」或「淘汰」，点击保存。

### 3.2 快捷入口
首页底部有「查看日报」和「查看周报」快捷按钮。

## 4. 数据管理

所有候选人的总览页面。管理员可看到**所有人的数据**。

### 4.1 筛选
- **部门/岗位/评估人**：下拉选择
- **日期范围**：选择起止日期
- 点击「清除」恢复显示全部

### 4.2 新增候选人
点击「新增候选人」，填写表单：
- 基本信息：姓名、手机、邮箱、性别、出生年、学校、专业
- **学历**（下拉）：本科（全日制）/ 专转本（全日制）/ 专转本（非全日制）/ 硕士 / 专科（全日制）/ 专科（非全日制）
- **应届生**（下拉）：是 / 否
- **招聘渠道**（下拉）：BOSS / 智联 / 前程无忧 / 猎聘 / 内推 / 校招线下双选会 / 实习僧 / 其他 / 新媒体（小红书、抖音等）
- **沟通时间**：日期选择器
- 部门 / 岗位 / 评估人 / 面试官

### 4.3 管道阶段
7 个勾选框追踪面试流程：
```
电筛通过 → 初试邀约 → 初试到面 → 初试通过 → 复试邀约 → 复试到面 → 复试通过
```
勾选对应的阶段即可，数据会自动反映到日报/周报统计中。

### 4.4 导入 Excel
支持批量导入 `.xlsx` 文件。第一行为表头，数据从第二行开始。列顺序：姓名、部门、岗位、评估人、面试官、沟通时间。

### 4.5 导出 CSV
点击「导出 CSV」下载当前筛选结果。

## 5. 推荐台账

显示所有 `result=推荐` 的候选人。

### 5.1 列说明
```
序号 | 岗位 | 实习生 | 沟通时间 | 姓名 | 手机 | 邮箱 | 性别 | 出生年 | 学校 | 专业 |
学历 | 是否为应届生 | 招聘渠道 | 推荐沟通情况 | 备注（特殊情况）
```

### 5.2 操作
- **双击**任意单元格 → 进入编辑模式 → 回车保存，Esc 取消
- **复制按钮**（📋）：将该行复制为 TSV 格式，可粘贴到 Excel
- **转换按钮**（⇄）：将该候选人转入淘汰台账
- **删除按钮**（🗑）：删除该候选人
- **填充模板**：在「推荐沟通情况」列点击模板图标，自动填入结构化沟通记录模板

### 5.3 筛选
- 搜索框：按姓名搜索
- 实习生下拉（管理员）：按实习生筛选
- 日期范围：按沟通时间筛选

### 5.4 导出
点击「导出 Excel」将当前筛选结果导出为 `.xlsx` 文件（含序号）。

## 7. 日报

### 7.1 新增行
点击「新增行」，系统自动拉取当天各渠道的电话沟通数。

列分组如下：
| 基础信息 | 开聊数 | 获取简历数 | 电话沟通数 | Moka | VIP账号权益使用 | 备注 |
|---------|--------|-----------|-----------|------|---------------|------|
| 日期、姓名 | Boss/智联/前程无忧 | Boss/智联/前程无忧/其他 | Boss/智联/前程无忧/Moka/其他 | 简历处理数 | Boss/智联/前程无忧 | 文字备注 |

### 7.2 刷新数据
点击某行的 🔄 按钮，重新从候选人表拉取当天通话数据并更新该行。

### 7.3 编辑
**双击**单元格进入编辑，回车保存。

### 7.4 导出
点击「导出 Excel」生成 `.xlsx` 文件。

### 7.5 管理员
管理员页面顶部有「填写人」下拉框，可切换到任意实习生的日报。

## 8. 周报

### 8.1 新增行
点击「新增行」，系统自动从候选人表计算：
- **各渠道沟通量/推荐量**：本周各渠道联系和推荐的候选人数量
- **初试/复试情况**：邀约数、到面数、通过数
- **面试率**：初试到面率、初试通过率、复试到面率、复试通过率（自动计算）

公式如下：
| 比率 | 说明 |
|------|------|
| 初试到面率 | 初试到面数 ÷ 初试邀约数 |
| 初试通过率 | 初试通过数 ÷ 初试到面数 |
| 复试到面率 | 复试到面数 ÷ 复试邀约数 |
| 复试通过率 | 复试通过数 ÷ 复试到面数 |

### 8.2 周总结 & 考勤
- **周总结**：一个文本框，自由填写
- **考勤情况**：一个文本框，填写日期/时间/原因

### 8.3 刷新数据
点击 🔄 按钮重新计算该行的管道数据和渠道统计数据。

### 8.4 管理员
管理员顶部有「填写人」下拉框。

## 9. 数据统计

显示**当日概览**，包含 7 个指标卡片，每个卡片显示：
- 今日值
- 昨日值
- 变化量（+/-）
- 环比百分比

可切换日期、部门、岗位筛选。

---

## 10. 用户管理（管理员）

路径：导航栏 → 用户管理

- **创建用户**：输入用户名、密码、显示姓名、角色
- **重置密码**：为任意用户设置新密码
- **启用/禁用**：切换用户激活状态

> 目前有两种角色：`admin`（管理员）和 `intern`（实习生）

## 11. 常见操作速查

| 操作 | 方法 |
|------|------|
| 上传简历 | 首页拖拽 PDF → 审核面板确认 → 保存 |

| 日期筛选 | 数据管理/台账的日期选择器 |
| 修改已录入的信息 | 双击单元格编辑 |
| 批量导入候选人 | 数据管理 → Excel 导入 |
